 HR Analytics Dashboard
 Project Overview:
This project presents an interactive HR Analytics Dashboard designed to provide actionable insights into workforce data. The dashboard enables stakeholders to monitor key HR metrics such as employee attrition, demographics, performance, and organizational trends to support data-driven decision-making.

Objectives:
Analyze employee data to identify attrition patterns
Track key HR KPIs across departments and roles
Provide insights into employee demographics and distribution
Enable management to make strategic workforce decisions

Tools & Technologies Used:-
Data Visualization: Power BI / Tableau / Excel (update accordingly)
Data Processing: Microsoft Excel
Data Cleaning: Excel / Power Query
Version Control: Git & GitHub

Dataset Information:
The dataset contains HR-related information such as:
Employee ID
Age, Gender, Department
Job Role & Experience
Salary / Compensation
Attrition Status
Performance Metrics.

Key Metrics & KPIs:
Employee Count
Attrition Rate
Average Salary
Department-wise Distribution
Gender Ratio
Job Satisfaction Levels
Dashboard Features:
1. Interactive filters (Department, Gender, Role, etc.)
2. Attrition analysis with trends
3. Employee demographic breakdown
4. Salary and performance insights
5. Visual KPIs for quick decision-making

Insights Derived:
->Identified departments with high attrition rates
->Observed correlation between salary and employee retention
->Highlighted key demographics influencing workforce trends
->Detected patterns in job satisfaction and performance
